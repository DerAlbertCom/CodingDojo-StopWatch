Coding Dojo - StopWatch by Albert Weinert
http://der-albert.com

A little StopWatch for Coding Dojos in Randori Style

Planed features

* List of Coders
* Timing
* Automatic Teambuilding
* Pausing Round Timing
* Displaying current Teammates
* Move a Coder to next round if they can make it to current round
* Standalone
* Visual Studio 2010 AddIn
